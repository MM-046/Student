# Data Analysis Project

## Overview
This project performs data processing, NumPy operations, and exploratory data analysis using Python.

## Features
- Data cleaning
- Missing value handling
- NumPy operations
- Data exploration using Pandas
- Statistical summary

## Technologies Used
- Python
- NumPy
- Pandas
- Jupyter Notebook

## Dataset
The dataset is stored in dataset.csv

## How to Run

Install libraries:

pip install -r requirements.txt

Run:

python data_processing.py
python numpy_operations.py
python pandas_exploration.py

## Output
The project generates insights and statistics from the dataset.
